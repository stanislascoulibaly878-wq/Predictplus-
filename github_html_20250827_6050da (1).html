Actions rapides

Déploiement sur GitHub Pages

1

Créer un compte GitHub

Si vous n'avez pas de compte, allez sur github.com et créez-en un.

2

Créer un nouveau dépôt

Cliquez sur le bouton "New repository" et nommez-le votre-nom-utilisateur.github.io (remplacez par votre nom d'utilisateur GitHub).

3

Télécharger les fichiers

Cliquez sur "Add file" → "Upload files" et téléchargez tous les fichiers de votre site web.

# Le fichier principal doit s'appeler index.html

4

Activer GitHub Pages

Allez dans Settings → Pages → Source, sélectionnez "main" branch et cliquez "Save".

5

Accéder à votre site

Votre site sera disponible à l'adresse https://votre-nom-utilisateur.github.io après quelques minutes.

Conseils supplémentaires

  • Utilisez un nom de domaine personnalisé si vous en avez un (dans les paramètres GitHub Pages)
  • Mettez à jour régulièrement votre site en téléchargeant les nouvelles versions des fichiers
  • Pour un site plus professionnel, envisagez d'utiliser Netlify ou Vercel qui offrent plus de fonctionnalités
